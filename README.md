# Sales Analytics Project

## Project Overview
This project analyzes retail sales data using Python to identify patterns, trends, and business insights.

## Tools Used
- Python
- Pandas
- Matplotlib

## Dataset
The dataset contains retail sales information including product name, category, region, sales, and profit.

## Analysis Performed
- Sales by Category
- Profit by Category
- profit by Region
- Sales by Region
- Top 10 Products by Sales
- Monthly Sales Trend

## Visualizations
Graphs were created to understand sales and profit distribution across different categories and regions.

## Conclusion
The analysis helps identify high-performing products, profitable categories, and strong sales regions.
